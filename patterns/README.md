# Patterns

Reusable design patterns for building cyborgenic organizations with [agent.ceo](https://agent.ceo).

Each pattern describes a recurring problem in multi-agent organizations, presents a proven solution, and includes practical code examples. Patterns are meant to be adapted to your specific needs — they are starting points, not rigid prescriptions.

## Index

- **[Multi-Agent Task Delegation](./multi-agent-task-delegation.md)** — Decompose complex work and delegate across specialized agents
- **[Knowledge-Base-Driven Agents](./knowledge-base-driven-agents.md)** — Give agents organizational context through shared knowledge
- **[Enterprise Discovery & Onboarding](./enterprise-discovery-onboarding.md)** — Discover and onboard the right agents for your enterprise
- **[Agent Meeting Protocol](./agent-meeting-protocol.md)** — Structured multi-agent communication for decisions and sync
