# Examples

End-to-end examples of cyborgenic organizations built with [agent.ceo](https://agent.ceo).

## Available Examples

| Example | Description | Agents |
|---------|-------------|--------|
| [Simple Org](./simple-org/) | Minimal 3-agent SaaS startup organization | CTO, Backend, Fullstack |

## Running an Example

1. Review the example's `org-config.yaml` to understand the organization structure
2. Customize agent roles and knowledge bases for your needs
3. Deploy using the agent.ceo dashboard or SDK

See each example's README for specific instructions.
